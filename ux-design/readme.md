The site map for Fitegy. Basically we will have 4 main screens: 
1. Timneline (feed)
2. Create
3. Challenge Manager
4. User Profile


## Wireframes

### Timeline Page 

<p align="center">
    <img src="https://github.com/software-students-fall2021/project-setup-team-fitegy/blob/master/ux-design/timeline.png?raw=true" width="500px" >
</p>

This is the timeline page of the app, where users can see their own posts as well as their friends' posts about the challenges they complete. Users can search, like, comment, check out the challenge or share the post. Users can access this page by clicking the home button on the bottom left.

<br>

### Create Page 

<p align="center">
    <img src="https://github.com/software-students-fall2021/project-setup-team-fitegy/blob/master/ux-design/create.png?raw=true" width="500px" >
</p>

This is the create page of the app, where users can create a challenge for themsevles, or compose a post by clicking on the post button. This page can be accessed by clicking the create button on the bottom left.  

<br>

### Create Challenge Page 

<p align="center">
    <img src="https://github.com/software-students-fall2021/project-setup-team-fitegy/blob/master/ux-design/create-details.png?raw=true" width="500px" >
</p>

This is the create challenge page of the app, where users can select options such as save, or save and send challenges to their friends. Users can type in their workout details, and include people in their challenge. 

<br>

### Challenge Manager Page 

<p align="center">
    <img src="https://github.com/software-students-fall2021/project-setup-team-fitegy/blob/master/ux-design/challenge-manager.png?raw=true" width="500px" >
</p>

This is the challenge manager page of the app, where users can manage all the ongoing or complete workout challenges they have made. This page can be accessed by clicking on the challenge manager button on the bottom right. 

<br>

## Update Challenge Page 

<p align="center">
    <img src="https://github.com/software-students-fall2021/project-setup-team-fitegy/blob/master/ux-design/update.png?raw=true" width="500px" >
</p>

This is the update challenge page of the app, where users can update all the ongoing, or complete workout challenges. This page can be accessed by clicking on the challenges from the challenge manager page.

<br>

## User Profile Page 

<p align="center">
    <img src="https://github.com/software-students-fall2021/project-setup-team-fitegy/blob/master/ux-design/profile.png?raw=true" width="500px" >
</p>

This is the user profile page of the app, where users can see a history of all the challenges they have completed, the number of people who participated with the user, the number of ongoing challenges, as well as any points or badges they earned.
